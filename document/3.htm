服务器配置文件

LinWin Http Server默认采用一种使用文本记述配置内容的方式，采用热更新机制， 能够做到修改完成即可直接生效。
内置的配置文件策略包括了服务器目录、服务端口、安全、API等。

网页服务配置

位于 {源代码目录}/config/Server.cfg 文件

内部记叙了服务的目录、端口、控制台登录、API接口。以下是默认的服务配置
注意：配置内容的 ':'符号必须与配置的参数空开一格，否则无法生效而且会错误。


Port: 10143;1
ServerDir: /var/www/html;2
ConsoleLogon_Port: 8888;3
API_Port: 10142;4

管理员用户配置文件

LinWin HttpServer不仅仅提供网络服务，而且可以使用用户来管理配置服务器。一个 用户由 账户 和 密码 构成，下面是默认的用户名与密码配置 （可修改）

提示：文件位于 {源代码目录}/config/Admin.cfg


User: linwin;Pwd: linwin;
User: admin;Pwd: admin;

默认页面配置文件

文件位于: {源代码目录}/config/defaultPage.cfg

LinWin Http Server支持默认页面的编辑，在默认值当中，index.html/index.htm等 将被索引。以下是默认的配置:


index.html
index.htm
index.php
index.jsp
